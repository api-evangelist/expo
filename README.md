# Expo

Expo is a React Native development platform providing cloud infrastructure for building, submitting, updating, and managing mobile apps. Expo Application Services (EAS) delivers cloud-driven APIs and CLI tooling for the full React Native app lifecycle.

## APIs

| Service | Description |
|---------|-------------|
| **EAS Build** | Cloud compilation and code-signing for Android and iOS React Native apps |
| **EAS Submit** | Automated upload to Apple App Store and Google Play Store |
| **EAS Update** | Over-the-air JavaScript bundle delivery to production users |
| **EAS Workflows** | CI/CD pipelines with cron scheduling and Slack integration |
| **EAS Hosting** | Edge deployment for Expo Router and React Native web apps |
| **Push Notifications** | Unified broker for Apple APNs and Google FCM push messages |
| **EAS Webhooks** | HMAC-SHA1 signed event callbacks for BUILD and SUBMIT events |

## Authentication

Programmatic access uses Personal Access Tokens or Robot User tokens set via the `EXPO_TOKEN` environment variable. Tokens take precedence over username/password credentials.
